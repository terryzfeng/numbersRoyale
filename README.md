# Numbers Royale

A PvP Royale Battle of Numbers

```
|=================================|
|          NUMBERS ROYALE         |
|=================================|
| P2: 1 points                    |
| Moves:                          |
|       1  2  3  4  5  _  7  8  9 |
| P1: 0 points                    |
| Moves:                          |
|       1  2  3  _  5  6  7  8  9 |
|=================================|
```


## Build

### Mac/Linux

To build and play

```
make
./numbersRoyale
```

To debug

```
make debug
```

To clean output files

```
make clean
```
